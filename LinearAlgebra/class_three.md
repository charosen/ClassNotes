## Class 3. 乘法与逆矩阵

<h3 id='3.'>一、乘法与逆矩阵</h3>

<h4 id='3.1.'>1. 方阵的逆</h4>

1. 定义：存在矩阵B，使得AB = BA = I(单位阵)，则称矩阵B为矩阵A的逆矩阵；
    + PS：对于方阵，左逆==右逆；对于非方阵，左逆$\not =$右逆；
2. 方阵可逆的充要条件：
    1. 定义法：单位矩阵的各列 = 原矩阵各列的线性组合，如果各列都在一条直线上，明显不可逆；
    2. 方阵的行列式不等于0；
    3. 方阵满秩；
    4. 方阵各列线性无关；
    5. 齐次线性方程组只有0解；
    6. 非齐次线性方程组有唯一解；
    7. 方阵特征值全不为0；

3. 逆矩阵的运算律：
    1. $(A^{-1})^{-1} = A$;
    2. $(kA)^{-1} = \frac{1}{k} A^{-1}$
    3. $ (A^T)^{-1} = (A^{-1})^T$;
    4. $(AB)^{-1} = B^{-1}A^{-1}$;
    5. 不服从分配律
